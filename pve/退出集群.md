# pve7 退出集群
- 停止cluster 服务
`systemctl stop pve-cluster.service`
`systemctl stop corosync.service`

- 设置本地模式
`pmxcfs -l`

- 删除corosync 配置文件
`rm /etc/pve/corosync.conf`
`rm -rf /etc/corosync/*`

- 重启cluster集群服务
`killall pmxcfs`
`systemctl start pve-cluster.service`

- 删除 WEB-UI 上的除当前 node 外的 node
`cd /etc/pve/nodes`
`rm -rf node_name`
